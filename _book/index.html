Introduction

This is a guide to help biologists at all levels analyze LC-MS/MS-based bottom-up proteomic data.

Included in this guide are:

  • Basic explanations of R and RStudio
  • Walk-through on each part of the proteomic data analysis pipeline including:
    • Data quality check
    • Data filtering
    • Handling missing values by imputation
    • Differential expression analysis using limma
    • Data visualization using ggplot2
