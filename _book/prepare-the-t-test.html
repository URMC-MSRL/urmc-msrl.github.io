4.2 Prepare the t-test

The first thing we need to do is design all the parameters for our t-test. In other words, declare what groups we want compared to each other.

4.2.1 Create the Comparison Table

#Create a list of all conditions
all_conditions <- imputed_data_df %>% 
  distinct(group) %>% 
  pull()

all_combinations <- as_tibble(t(combn(all_conditions, m = 2)),
                              .name_repair = ~ make.names(., unique = T)) %>% 
  rename(V1 = X, V2 = X.1) %>%

               names_to = 'name',

  group_by(group) %>% 
  mutate(comparison = list(combinations)) %>%

This data frame is made up of two columns:

  • group: A string denoting the condition, or group (in this case, the E. coli genotype)

  • comparison: a list made up of all the possible group comparison for the specific group, with each group separated by “vs”.

    • For example, the group WT contains the following comparisons:

      • “WT_vs_bisC”

      • “WT_vs_msrA”

      • “WT_vs_msrB”

      • “WT_vs_msrC”

    • While the group bisC contains:

      • “WT_vs_bisC”

      • “bisC_vs_msrA”

      • “bisC_vs_msrB”

      • “bisC_vs_msrC”

    • And so on for the other 3 genotypes

4.2.2 Prepare Our Data Using the Comparison Table

We are nor almost ready to perform a t-test.

Recall that we will be using our data frame containing the imputed values. Before going any further, we need to add a new column to that data frame: a comlumn containing replicate number

imputed_data_df <- imputed_data_df %>% 
  mutate(rep = as.integer(str_sub(Run, -1)))

Now that our data frame is ready, we will move it around so it’s in a form that is full ready for limma.

The main we need to do for this is to each each comparison we want to have evaluated.

ttest_data_prep <- imputed_data_df %>%
  ungroup() %>%
  distinct(Protein.Group, group, rep, PG.MaxLFQ) %>%
  complete(nesting(group, rep),
           Protein.Group) %>%
  group_by(Protein.Group, group) %>%
  mutate(n_detect = sum(!is.na(PG.MaxLFQ))) %>%
  mutate(n_replicates = n()) %>%
  ungroup() %>%
  left_join(all_combinations, by = 'group') %>%
  unnest(comparison) %>% 
  unite(group:rep, col = 'Run', sep = '_', remove = FALSE)

Important Note

You might have noticed that I had to re-order our group column. Due to how we design the contract matrix, it is imperative that we have the groups column be in alphabetical order (regardless of case-type). Before proceeding, make sure this is done through whatever is necessary.

In this scenario, R sorts alphabetically by putting capital letters first, regardless of position in the alphabet. To get around this, we first made WT all lowercase, then sorted group to get WT last, and then replaced it again with capital letters to match the rest of our metadata.

moderated_t_test_design <- model.matrix(
  ~ 0 + factor(str_replace_all(pull(moderated_t_test_map, group),
                               pattern = ' ', replacement = '_'))
)

colnames(moderated_t_test_design) <- levels(
  factor(str_replace_all(pull(moderated_t_test_map, group),
                         pattern = ' ',
                         replacement = '_'))
)

4.2.4 Fit the T-Test Regression Models

# Fit the regression model
moderated_t_test_fit <- suppressWarnings(lmFit(moderated_t_test_input,
                                               moderated_t_test_design))
# Create a matrix of differences
names <- paste0(
  'x', 
  str_extract(unique(pull(comparison_data, comparison)),
              pattern = '.+(?=_vs_)'), 
  '_vs_x',
  str_extract(unique(pull(comparison_data, comparison)),
              pattern = '(?<=_vs_).+')
  )
