We will also make a new data frame from our imputed data frame that we will be using across all of our replicate comparison plots. This will already be in a format more amenable to plotting so we won’t need to perform all the same transformations multiple times before drawing every plot.

sample_level <- ttest_data_prep_wt_comps %>% 
  dplyr::select(-c(group, rep, n_detect, n_replicates, comparison)) %>% 
  mutate(Run = gsub('_', '', Run)) %>% 
  left_join(metadata %>% 
              dplyr::select(Run, Condition, Replicate),
            by = 'Run') %>% 
  set_names('sample', 'protein_group', 'log2_abundance', 'group', 'rep') %>% 
  group_by(protein_group, group) %>% 
  mutate(mean_log2_abundance = mean(log2_abundance)) %>% 
  mutate(abundance = 2^log2_abundance, .after = log2_abundance) %>% 
  left_join(imputed_data_wide_df %>% 
              distinct(Protein.Group, Genes) %>% 
              set_names('protein_group', 'gene_name'),
            by = 'protein_group') %>% 
  left_join(protein_description %>% 
              distinct(Gene, Description) %>% 
              set_names('gene_name', 'protein_name'),
            by = 'gene_name') %>% 
  relocate(gene_name:protein_name, .after = protein_group) %>% 
  relocate(protein_group:protein_name, .before = 'sample') %>% 
  relocate(group:rep, .after = 'sample') %>% 
  ungroup()

We will also make a new data frame from out t-test results that we will be using across all of our differential expression plots for similar reasons as above.

diff_expression_level <- ttest_data %>% 
  dplyr::rename(protein_group = 'Protein.Group') %>% 
  left_join(y = sample_level %>% 
              ungroup() %>% 
              distinct(protein_group, gene_name, protein_name),
            by = 'protein_group') %>% 
  relocate(gene_name:protein_name, .after = protein_group) %>% 
  relocate(comparison, .after = protein_name) %>% 
  mutate(comparison = sub('_vs_', '/', comparison))

5.1.1 Make a Common Plotting Theme

Lastly, we will make a common theme for our plots and saving them to a new object called plot_theme. We can then apply these themes to all of our plots by simply calling plot_theme instead of typing all these theming components for each plot.

plot_theme <- theme_bw() +
   theme(plot.title   = element_text(size = 20),
         axis.title.x = element_text(size = 15),
         axis.text.y  = element_text(size = 15),
         axis.text.x  = element_text(size  = 12,
                                     angle = 75,
                                     hjust = 1),
         axis.title.y = element_text(size = 15),
         legend.title = element_text(size = 15),
         legend.text  = element_text(size = 15))
